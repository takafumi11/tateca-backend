Remove an unjoined member from a group

Removes an unjoined member from the specified group. Only unjoined members (those not yet linked to an authenticated account) can be removed.

Authorization: The requester must be an authenticated member of the group. Any authenticated group member can remove unjoined members — no special privileges required.

Restriction: Joined members (those linked to an authenticated account) cannot be removed. Joined members must leave the group voluntarily via the leave-group endpoint.

BearerAuth

required
string <uuid>
